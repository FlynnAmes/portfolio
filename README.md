<h1> Overview </h1>

This GitHub repository is a work-in-progress portfolio, containing projects of various sorts.
The first project demonstrates deployment of a machine learning model for predicting credit risk.
See embedded README's for more detail.

More to come.

<h3> Links to published work (for those interested) </h3>

- <i>Machine learning for weather prediction </i> - a proof of concept for hyper-local forecasts using crowdsourced weather observations. Publication <a href=https://doi.org/10.1002/met.2200>here</a>, and news summary <a href=https://www.metoffice.gov.uk/about-us/news-and-media/media-centre/weather-and-climate-news/2024/urban-heat-forecasts-could-be-enhanced-with-machine-learning>here</a>. 

- <i>Numerical modelling of icy moon oceans </i> - Adapting (physics-based) Earth ocean models to simulate the physical oceanography of Enceladus (moon of Saturn), showing it to be an important consideration for future missions aiming to find life there. Publication  <a href=https://www.nature.com/articles/s43247-025-02036-3>here</a> and news summary <a href=https://thedebrief.org/researchers-pour-cold-water-on-attempts-to-detect-aqueous-alien-life-on-enceladus-as-new-difficulties-emerge/>here</a>.
